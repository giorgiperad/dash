# ALKDots Font Setup Instructions

To use the ALKDots bold font for Georgian text, please follow these steps:

## Download the Font

1. Visit https://web-fonts.ge/alkdots/
2. Click on "გადმოწერა" (Download) button
3. Extract the downloaded archive

## Install Font Files

Copy the following font files to the `public/fonts/` directory:
- `alkdots-bold.woff2`
- `alkdots-bold.woff`
- `alkdots-bold.ttf`

If the files have different names in the archive, rename them to:
- `alkdots-bold.woff2`
- `alkdots-bold.woff`
- `alkdots-bold.ttf`

## Alternative: Use CDN

If you prefer to use the CDN instead, you can download the font files from:
- https://web-fonts.ge/alkdots/

And update the `@font-face` declaration in `index.html` to point to the correct CDN URLs.

## Font Usage

The font is already configured in the CSS and will be applied to the Georgian quote text:
"წარმატება მოდის იმათთან, ვინც დარჩა, როცა სხვები წავიდნენ!"
